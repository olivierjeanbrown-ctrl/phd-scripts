
# Literature Review Tools & Templates


Within this directory is a series of excel spreadsheets used as guideline templates to assist with literature reviews.

---

## Lit_Review_Template

- Specific search strategy template used for a systematic literature review on pediatric concussion.

## Searching_log_pediatric_concussion

- Advanced version of Lit_Review_Template, with notes and logs to track identified reports.

## Systematic_Search_Strategy_Template
- Search strategy with implemented search order queues for a review on neuroimaging literature pertaining to pediatric TBI.
